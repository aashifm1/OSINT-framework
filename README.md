# OSINT Framework 


<img width="1920" height="1080" alt="osint" src="https://github.com/user-attachments/assets/8ab650da-c2e2-4877-8f31-0995ad09af79" />



Advanced open-source intelligence reconnaissance tool with async scanning, 80+ platforms, and structured reporting.

> **Legal:** For authorized, ethical, and lawful use only. Unauthorized use may violate CFAA, GDPR, CCPA, and local laws.


## Requirements

```bash
pip install -r dependency.txt
```

Python 3.10+ required.


## Usage

```bash
python osint.py
```

The tool walks you through an interactive prompt — no flags needed.


## Features

| Feature | Details |
|---|---|
| Platform scan | 80+ platforms across 10 categories |
| Speed | Async engine, 20 concurrent requests |
| Username variants | Leet, suffixes, name-derived handles |
| Email intel | Gravatar hash, MX records, HIBP stub |
| DNS / WHOIS | A, MX, NS, TXT records + registrar info |
| Dork generation | 8 categories, opens in browser |
| Exposure score | 0–100 CVSS-inspired risk rating |
| Reports | JSON, CSV, TXT, HTML (dark-mode dashboard) |
| Resume support | Checkpoint saves progress on crash |
